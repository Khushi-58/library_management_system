Library Management System (LMS) – Frontend

A simple web-based Library Management System frontend that allows users to view available books, borrow books, and return borrowed books. This project demonstrates a clean UI structure using HTML, CSS, and JavaScript.

Project Overview
The Library Management System frontend provides a simple interface for users to:
View all available books
Borrow books
Return borrowed books
Navigate between pages using a clean and simple UI

This frontend can be linked with a backend for full functionality (CRUD operations, database management, and user authentication).

Features
Homepage with welcome message and navigation links
Book listing page with search/filter functionality
Borrow page for selecting and borrowing books
Return page for returning borrowed books
Responsive navigation and simple styling

Pages & Description

Page	    Description
index.html	Homepage: Welcome message, login/register buttons, navigation links.
books.html	Book listing: Displays all available books and search/filter functionality.
borrow.html	Borrow book page: Users can select books to borrow and confirm.
return.html	Return book page: Users can return borrowed books.
